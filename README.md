# 🧶 QA_Banquinet: Code_Weaver Edition

> **"El conocimiento es un ciudadano de primera clase."**

Este repositorio es una base de conocimientos activa y un framework de testing para el ecosistema **gPOS**. Aquí, el código (checks de tests, matrices) y el conocimiento (reglas de negocio, manuales) están entrelazados para permitir una colaboración fluida entre humanos y agentes de IA.

## ✨ Filosofía Code_Weaver
- **Simbiosis Humano-IA:** Documentamos para que la IA entienda el contexto y la IA nos ayuda a mantener la trazabilidad.
- **Obsidian Vault:** Toda la carpeta `docs/` está optimizada para ser navegada en [Obsidian](https://obsididan.md) usando [[wikilinks]], tags y Mapas de Contenido (MOC).
- **Documentación Viva:** Si algo cambia en el sistema, se refleja en el `DEV_LOG.md` y se actualiza en la base de conocimientos.

## 🧠 Estructura del "Cerebro"
- **[[00_KNOWLEDGE_BASE_MOC|🗺️ Mapa de Contenido (MOC)]]**: El punto de entrada principal.
- **[[DEV_LOG|🕒 Log de Desarrollo]]**: Decisiones y progreso diario.
- **[[00_INBOX/README|📥 Bandeja de Entrada]]**: Para capturar ideas rápidas sin fricción.

### 📂 Carpetas Clave
- `docs/core/`: Visión, arquitectura y principios fundamentales.
- `docs/use_cases/`: Casos de uso formales (Gherkin/Markdown).
- `docs/concepts/`: Reglas de negocio específicas por juego (Quiniela, 5 de Oro, etc.).
- `docs/ai_agents/`: Mandamientos y reglas para los agentes de IA.

## 🛠️ Herramientas y Metodología

### Los 7 Mandamientos de QA (TL;DR)
1. Títulos concisos (< 10 palabras).
2. Certeza absoluta (cero ambigüedad).
3. Hechos vs Opiniones.
4. Contraste Actual vs Esperado.
5. Anexos con identidad.
6. Verificación de Regresión.
7. Pasos reproducibles para cualquier persona.
