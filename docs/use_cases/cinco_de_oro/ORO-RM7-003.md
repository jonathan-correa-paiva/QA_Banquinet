# ORO-RM7-003: Jugada de 7 Números Revancha sorpresa

## Descripción y Escenario

**ID:** ORO-RM7-003
**Título:** Ingresar una múltiple de 7 números sorpresa con modalidad Revancha.
**Tags:** #automatizado #5deoro #revancha #sorpresa #multiple

### Escenario (Gherkin)

**DADO** que se activa "Revancha" y se elige "Múltiple de 7"
**CUANDO** presiona "Sorpresa"
**ENTONCES** genera 7 números y agrega al carrito el equivalente a 21 apuestas dobles (Oro + Revancha).
