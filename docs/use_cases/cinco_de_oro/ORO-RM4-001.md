# ORO-RM4-001: Jugada de 4 Números Revancha sorpresa

## Descripción y Escenario

**ID:** ORO-RM4-001
**Título:** Ingresar una múltiple de 4 números sorpresa con modalidad Revancha.
**Tags:** #automatizado #5deoro #revancha #sorpresa #multiple

### Escenario (Gherkin)

**DADO** que se activa "Revancha" y se elige "Múltiple de 4"
**CUANDO** presiona "Sorpresa" y completa el 5to número
**ENTONCES** agrega al carrito el equivalente a 44 apuestas dobles (Oro + Revancha).
