# ORO-FR5-001: Jugada de 5 Números Revancha fija

## Descripción y Escenario

**ID:** ORO-FR5-001
**Título:** Ingresar una jugada de 5 números manual (Fija) con la modalidad Revancha.
**Tags:** #automatizado #5deoro #revancha

### Escenario (Gherkin)

**DADO** que el subagente activa "Revancha"
**CUANDO** ingresa manualmente los 5 números (ej: 01-02-03-04-05)
**ENTONCES** el sistema valida la jugada y la ingresa al carrito como una jugada de 5 de Oro + Revancha.
