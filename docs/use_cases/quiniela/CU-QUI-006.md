# CU-QUI-006: Combinación de Sorteos (Quiniela)
**Módulo:** Quiniela 

## 📝 Descripción
**Valor de Negocio:** Permitir flexibilidad al apostador para elegir múltiples turnos en una misma transacción.

---

## ✅ Comportamiento Esperado
* Verificar que genera distintos tickets según los sorteos seleccionados.
