# CU-CIE-001: Antes de las 14hs: ambos sorteos (Vesp y Noct) están apostables
**Módulo:** Cierres 

## 📝 Descripción
**Valor de Negocio:** Asegurar que durante el horario operativo matutino, el sistema permita la venta para ambos sorteos principales del día.

---

## ✅ Comportamiento Esperado
* El cierre muestra Vesp y Noct disponibles.
* Se puede apostar a ambos sorteos.
