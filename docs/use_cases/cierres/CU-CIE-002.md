# CU-CIE-002: Pasadas las 14hs: solo el sorteo Nocturno queda apostable
**Módulo:** Cierres 

## 📝 Descripción
**Valor de Negocio:** Garantizar el corte automático del sorteo Vespertino para evitar apuestas fuera de hora.

---

## ✅ Comportamiento Esperado
* El Vespertino desaparece de los sorteos disponibles.
* Solo se puede apostar al Nocturno.
